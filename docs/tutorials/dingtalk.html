钉钉配置教程

7 个步骤 · 难度：中等

在钉钉开放平台创建智能机器人，接入 PackClaw，通过钉钉与 AI 对话。

教程概览

完成以下步骤，你就能让 PackClaw 接入钉钉：

  1. 进入钉钉开放平台
  2. 创建应用
  3. 添加机器人能力
  4. 配置权限并发布版本
  5. 获取 Client ID 和 Client Secret
  6. 在 PackClaw 中填写凭证
  7. 在钉钉中找到机器人

步骤 1: 进入钉钉开放平台

在浏览器中访问 open.dingtalk.com，使用钉钉扫码登录。

步骤 2: 创建应用

进入 「应用开发」 → 「机器人」，点击 「创建应用」。

步骤 3: 添加机器人能力

点击 「添加应用能力」，选择 「机器人」。

步骤 4: 配置权限并发布版本

进入 「权限管理」，在搜索框中输入「消息」，然后 「批量申请」 消息相关权限。

进入 「版本管理与发布」，点击 「创建新版本」。

步骤 5: 获取 Client ID 和 Client Secret

进入 「凭证与基础信息」 页面，复制 Client ID 和 Client Secret。

步骤 6: 在 PackClaw 中填写凭证

打开 PackClaw，进入 「设置」 → 「聊天集成」，开启钉钉，将 Client ID 和 Client Secret 填入对应输入框。

步骤 7: 在钉钉中找到机器人

打开钉钉，搜索你创建的机器人名称，找到机器人并发送一条消息。

钉钉配置完成！

你的钉钉机器人已经创建并接入 PackClaw，现在可以通过钉钉与 AI 对话了。
