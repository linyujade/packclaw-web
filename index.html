还有疑问？

内置的免费额度能用多久？
▾
免费额度约可完成 100~200 次对话，足够体验所有内置场景。额度用完后，可在 PackClaw 设置页一键充值，或者切换成自己的 Anthropic / Kimi 等 API Key 继续使用，非常灵活。
PackClaw 和 OneClaw 有什么区别？
▾
OneClaw 是 OpenClaw 的一键安装包，需要用户自行配置 API Key。PackClaw 在此基础上更进一步：内置免费额度（无需 Key）+ 10 个预置场景（开箱即用）+ 更深度的中国本土场景适配。
需要科学上网吗？
▾
不需要。PackClaw 内置国内镜像源，模型请求走国内中转节点，所有功能在大陆网络环境下均可正常使用，无需任何代理工具。
我的数据安全吗？
▾
PackClaw 的核心逻辑运行在你的本地电脑上。对话内容通过 HTTPS 加密传输到 AI 模型，不会被 PackClaw 储存或分析。文件读写操作仅在你本地磁盘进行。
安装后 AI 没有响应怎么办？
▾
请检查网络连接是否正常。若内置免费额度已用完，页面会有明显提示，按提示充值或切换 Key 即可恢复。如果仍有问题，可以通过应用内的「反馈」按钮联系我们。
